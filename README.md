# FolioVault

FolioVault is a personal portfolio vault built with React, TypeScript, Vite, CSS, and `@chenglou/pretext`.

It helps you store and manage projects, activities, certificates, awards, study records, and retrospectives as portfolio cards in a responsive Masonry board.

## Features

- Initial sample portfolio cards
- Add, edit, and delete cards
- localStorage persistence
- Search, category filter, status filter, tag filter, and featured-only filter
- Sorting by recently updated, newest, oldest, title, and featured-first
- Right-side detail panel
- Responsive absolute-positioned Masonry layout
- Summary text height estimation with `@chenglou/pretext` `prepare()` and `layout()`
- Thumbnail image URL or local image upload
- JSON backup and restore

## Tech Stack

- React
- TypeScript
- Vite
- @chenglou/pretext
- CSS

## Getting Started

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite.

## Build

```bash
npm run build
```
